Si format == NULL :
    retourner -1

Initialiser la liste d’arguments
Initialiser count = 0
Initialiser i = 0

Tant que format[i] != '\0' :

    si format[i] n'est pas '%' :
        afficher format[i]
        incrémenter count
        avancer à l’index suivant
        continuer la boucle

    sinon (format[i] == '%') :
        avancer à i+1
        lire le spécificateur

        obtenir la fonction correspondant au spécificateur

        si fonction trouvée :
            appeler la fonction avec les arguments
            ajouter au count le nombre de caractères imprimés

        sinon :
            afficher '%'
            afficher le spécificateur inconnu
            count += 2

    avancer à l’index suivant

Terminer la liste d’arguments
Retourner count
